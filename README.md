# e-plantShopping

## Paradise Nursery Shopping Application

This project is a React-based shopping application for Paradise Nursery. Users can browse a variety of houseplants, add them to a shopping cart, update item quantities, remove items, and view the total cost before checkout.

### Features

- Landing page with company information
- Product listing page with categorized houseplants
- Shopping cart with quantity controls
- Dynamic cart item counter
- Redux state management
- React Router navigation
- Responsive user interface

### Technologies Used

- React
- Redux Toolkit
- React Router
- CSS
- GitHub Pages
